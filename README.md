<h2>Torquise</h2>

1.Run git clone <my-cool-project>
    
2.Run composer install

3.Run cp .env.example .env
    
4.Run php artisan key:generate
    
5.Run php artisan migrate
    
6.Run php artisan serve

7.Go to link localhost:8000
